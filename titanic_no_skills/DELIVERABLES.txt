🎯 KEY FEATURES OF SOLUTION
══════════════════════════════════════════════════════════════════════════

1. APPROACH
   • Statistical probability-based model
   • No machine learning libraries required
   • Fully transparent and interpretable
   • Based on historical patterns in data

2. FEATURES USED
   • Passenger Class (Pclass)
   • Sex/Gender
   • Age (with intelligent missing value handling)
   • Fare (with intelligent missing value handling)
   • Embarked port (secondary)

3. KEY INSIGHTS CAPTURED
   • Women and children first principle (96-97% female survival in 1st class)
   • Class-based access to lifeboats
   • Age-related survival advantage for children
   • Wealth/fare correlation with survival

4. PREDICTION ADJUSTMENTS
   • Children < 10 years: +50% survival boost
   • Teens 10-18 years: +20% survival boost
   • Elderly > 60 years: -30% survival penalty
   • High fare passengers: +20% boost
   • Low fare passengers: -20% penalty
