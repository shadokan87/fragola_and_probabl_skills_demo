PROBLEM: Predict which Titanic passengers survived
DATASET: 891 training samples, 418 test samples
APPROACH: Statistical probability model
LANGUAGES: Python 3 (no external ML libraries needed)
TARGET: Binary classification (0 = Died, 1 = Survived)

┃ 🎯 KEY FINDINGS FROM DATA ANALYSIS                                      ┃
┗━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━┛

Finding #1: Sex is the Strongest Predictor
────────────────────────────────────────
Female 1st Class:  96.81% survival (91/94)      │ Men had much lower
Female 2nd Class:  92.11% survival (70/76)      │ survival rates across
Female 3rd Class:  50.00% survival (72/144)     │ all classes due to
                                                  │ "women and children
Male 1st Class:    36.89% survival (45/122)     │ first" principle
Male 2nd Class:    15.74% survival (17/108)     │
Male 3rd Class:    13.54% survival (47/347)     │

Finding #2: Passenger Class Matters
──────────────────────────────────
1st Class: Best access to lifeboats and information
2nd Class: Moderate survival rates
3rd Class: Limited access and overcrowding

Finding #3: Age Affects Survival
────────────────────────────────
Children (<10 yrs):  High survival (protected by "children first")
Teens (10-18 yrs):   Moderate survival
Young Adults (18-35): Depends on sex and class
Elderly (>60 yrs):   Lower survival rates

Finding #4: Fare Correlates with Survival
────────────────────────────────────────
Higher fare = Better cabin location = Closer to lifeboats
Also often correlated with 1st class tickets

┃ 🛠️  ALGORITHM EXPLANATION                                              ┃
┗━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━┛

Step 1: Calculate baseline survival probability
──────────────────────────────────────────────
For each (Sex, Class) combination:
  baseline_prob = survivors_in_group / total_in_group

Step 2: Adjust based on individual characteristics
───────────────────────────────────────────────────
  IF age < 10:      multiply by 1.5 (children priority)
  IF age 10-18:     multiply by 1.2 (youth advantage)
  IF age > 60:      multiply by 0.7 (elderly disadvantage)
  IF fare > 1.5×median:  multiply by 1.2 (wealth advantage)
  IF fare < 0.5×median:  multiply by 0.8 (poverty disadvantage)

Step 3: Make prediction
────────────────────────
  IF adjusted_probability > 0.5:  Survived = 1
  ELSE:                            Survived = 0
